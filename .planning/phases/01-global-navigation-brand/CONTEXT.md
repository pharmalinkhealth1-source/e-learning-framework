# Phase 1: Global Navigation & Brand

## Goal
Implement the core design system tokens (HDS) and the global navigation shell with animated megamenus and dark mode support.

## Requirements
- **REQ-NAV-01**: HDS Design Tokens (CSS Variables).
- **REQ-NAV-02**: Sticky Global Navigation with backdrop blur.
- **REQ-NAV-03**: Animated Megamenus (Hover-reveal).
- **REQ-NAV-04**: Dark Mode Toggle support.
- **REQ-NAV-05**: Micro-interactions & Spring Animations.

## Context
- **Design System**: Stripe HDS (Heading Design System) parity.
- **Stack**: Next.js 15, Vanilla CSS Modules, Framer Motion.
- **Base Style**: Clinical Luxury / High-Density.

## Success Criteria
1. Global CSS contains HDS tokens for colors, spacing, and typography.
2. Navigation bar is responsive and sticky.
3. Megamenus reveal smoothly on hover using Framer Motion.
4. Dark mode toggle correctly updates all design tokens.
