# Flip Card Carousel - User Manual

## 🚀 Getting Started

You have two widgets available in Elementor:
1. **Flip Card Carousel** (Manual) - Best for a few static cards.
2. **Flip Card Carousel (CPT)** (Dynamic) - Best for managing many cards like "Success Stories".

---

## 1️⃣ using the Dynamic Version (Recommended)

This version pulls data from the "Success Stories" section in your dashboard.

### Step 1: Add Content
1. Go to your WordPress Dashboard.
2. Click on **Success Stories** (in the left menu) → **Add New**.
3. Fill in the **Story Details** box:
   - **Person Name**: e.g. "Dr. Sarah Smith"
   - **Job Title**: e.g. "Head Pharmacist"
   - **Quote**: Short testimonial.
   - **Description**: The full story.
   - **Story Link**: (Optional) Link to their full bio page.
4. Set the **Featured Image** (right sidebar) - This is the profile photo.
5. Click **Publish**.

### Step 2: Add to Page
1. Edit your page with **Elementor**.
2. Search for widget: **Flip Card Carousel (CPT)**.
3. Drag it onto the page.
4. It will automatically show your published stories!

### Step 3: Settings
- **Query Settings**: Change number of cards, sort order.
- **Style Tab**: Customize colors, fonts, and card size.

---

## 2️⃣ Using the Manual Version

This version lets you type content directly into Elementor.

### Step 1: Add to Page
1. Edit your page with **Elementor**.
2. Search for widget: **Flip Card Carousel** (the one without "CPT").
3. Drag it onto the page.

### Step 2: Add Cards
1. In the **Content** tab, look for **Cards**.
2. Click **Add Item**.
3. Fill in the details (Name, Title, Quote, Image, etc.).
4. Repeat for other cards.

---

## ❓ Troubleshooting

- **Cards not showing?** Make sure you have Published stories (not Drafts).
- **Images missing?** Ensure "Featured Image" is set for each story.
- **Styling issues?** Try regenerating CSS in Elementor > Tools.
