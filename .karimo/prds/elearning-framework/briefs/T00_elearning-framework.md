1. From the project root, run:
   ```

2. Confirm the command exits with code 0 and no unresolved peer dependency warnings appear in stdout. If peer warnings appear, resolve them (e.g., with `--legacy-peer-deps` only if explicitly needed and documented in a comment in `package.json`).

4. Run `npx tsc --noEmit` and confirm it passes with no new errors.
5. Commit `package.json` and `package-lock.json` together with message:
   ```
